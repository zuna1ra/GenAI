# Local LLM Chatbot

A Streamlit-based chatbot that connects to locally hosted LLMs using Ollama.

## Features

- Chat interface using Streamlit
- Multiple model selection
- Temperature control
- Chat history
- Download conversations
- Local AI inference with Ollama

## Technologies Used

- Python
- Streamlit
- Ollama
- Requests

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
